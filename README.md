# Server-in-rust

A lightweight, local server implementation written in Rust. This project focuses on simple server-side logic and networking basics.

## Features

* **Local Execution:** Designed for individual use and local testing.
* **Rust-powered:** Built with memory safety and performance in mind.

## Prerequisites

* [Rust](https://www.rust-lang.org/tools/install) installed on your system.

## Getting Started

1. **Clone the repository:**
```bash
git clone https://github.com/Shivanshu-Singh-2203/Server-in-rust.git
cd Server-in-rust

```

2. **Run the server:**
```bash
cargo run

```

## Usage

Simply start the server and interact with it using your preferred terminal or browser tools (like `curl` or `Postman`).
